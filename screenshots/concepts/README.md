# Concept Previews

Concept previews are visual direction references for SableOS. They are not final UI screenshots and should not imply that every visual feature is implemented.

Use this folder for lightweight concept images that communicate the intended feel of SableOS, such as command hub layouts, Ship Mode previews, config-driven mode concepts, or future desktop mode switching ideas.

## Guidelines

- Keep images lightweight, ideally under 1 MB each.
- Avoid private information, secrets, personal desktop details, student or school information, private paths, account names, messages, calendars, emails, and other personal data.
- Use relative paths when linking images from documentation.
- Separate actual screenshots from concept previews. Real screenshots should remain in the broader `screenshots/` area or another clearly labeled screenshot folder.
- Label concept imagery clearly so readers understand it represents visual direction, not finished UI.
