# SableOS v0.2.0

Sable CLI release.

## Summary

SableOS v0.2.0 adds the first version of Sable CLI, the main command hub for the SableOS environment.

This release gives SableOS a central command interface through the `sable` command, allowing users to check status, list modules, run Ship Doctor, route installs, and access documentation paths from one place.

## Highlights

- Added Sable CLI as the main command hub
- Added the `sable` command
- Added `sable help`
- Added `sable version`
- Added `sable modules`
- Added `sable doctor`
- Added `sable status`
- Added `sable docs`
- Added `sable install` routing
- Added Sable CLI installer
- Updated the root installer to support Sable CLI
- Fixed Ship Doctor repository detection when installed
- Updated SableOS module documentation
