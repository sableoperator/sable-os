# SableOS v0.2.1

CLI command rename patch.

## Summary

SableOS v0.2.1 renames the installed SableOS CLI command from `sable` to `sableos`.

This avoids a command name conflict with the existing Sable AI Core command already used on the BEBOP workstation.

## Why This Patch Exists

SableOS v0.2.0 introduced the first Sable CLI command hub using the command name:

sable
