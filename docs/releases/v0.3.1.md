# SableOS v0.3.1

Ship Mode language cleanup release.

## Summary

SableOS v0.3.1 generalizes Ship Mode language so the public project is less specific to one personal workflow.

This patch keeps Ship Mode preview-only while making the default mode descriptions more useful for a wider audience.

## What Changed

The default Ship Mode descriptions were updated to describe general desktop states instead of one specific workstation routine.

## Updated Ship Mode Language

### Work

Focused productivity, planning, communication, writing, task management, and deep work.

### Lounge

Relaxed desktop use, media, browsing, ambient use, personal time, and low-pressure creative work.

### Clean

A minimal reset mode designed to reduce visual clutter and return the desktop to a quiet baseline.

### Engineering

Technical work, development, diagnostics, system tools, local services, logs, and build workflows.

## Safety

This release does not add real desktop switching.

Ship Mode remains preview-only and does not change:

* Wallpapers
* Widgets
* Startup apps
* Cinnamon settings
* Panels
* Windows
* Displays
* Running processes

## Changed

* Generalized Ship Mode descriptions
* Updated preview output language
* Removed overly personal workflow references from public Ship Mode wording
* Updated version to `0.3.1`

## Known Limitations

* Ship Mode is still preview-only
* No mode config files yet
* No real desktop switching yet
* No backup or rollback system for desktop changes yet
* No uninstall flow yet

## Next Goals

* Add Ship Mode config files
* Add dry-run action plans
* Add backup and rollback support before any real desktop changes
* Prepare the path toward safe mode switching in a future release
