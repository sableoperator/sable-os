# SableOS v0.1.0

First installable module release.

## Summary

SableOS v0.1.0 establishes the first working foundation of the project.

This release includes the initial project structure, documentation system, module registry, Git workflow, install plan, and the first installable module: Ship Doctor.

## Highlights

- Created the SableOS project repository
- Added project README
- Added project charter
- Added roadmap
- Added module registry
- Added documentation index
- Added system structure documentation
- Added screenshot safety guide
- Added Git workflow documentation
- Added install plan
- Added MIT license
- Added Ship Doctor as the first official module
- Added Ship Doctor installer
- Added root SableOS installer

## Ship Doctor

Ship Doctor is the first installable SableOS module.

It provides local system diagnostics, including:

- System information
- Disk usage
- Memory availability
- Monitor detection
- Ollama/local AI availability
- SableOS project structure checks
- Backup freshness checks
- Warning and failure status summaries
